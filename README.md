# lita-clever

`lita-clever` is a handler for [Lita](https://github.com/jimmycuadra/lita) that integrates Cleverbot.

## Installation

Add lita-clever to your Gemfile:

``` ruby
gem 'lita-clever'
```

## Configuration

No Configuration.

## Usage

```
Lita: lita hello
> Hello
```
